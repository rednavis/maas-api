# maas-api
Server side API of the Material Assets management system.

##Build project
`./gradlew clean build`

`./gradlew bootRun`

##Create dependency report
`./gradlew clean htmlDependencyReport`

##Create docker image
`./gradlew clean bootJar jibDockerBuild`

##Simpe controller
http://localhost:8081/shared
